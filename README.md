El Proyecto-contactos-alkemy es un repositorio de GitHub creado por Mona Jensen que contiene una aplicación de gestión de contactos. El proyecto está compuesto por cuatro archivos Python que implementan un sistema de gestión de contactos. Se trata de un proyecto público con un único compromiso inicial que carga los archivos del proyecto. Actualmente no cuenta con documentación README, descripción detallada ni temas asociados.

Puntos Funcionales Principales
Gestión y almacenamiento de contactos (contacto_final.py)
Sistema de gestión de contactos (gestión_final.py)
Interfaz principal de la aplicación (main_final.py)
Módulo de pruebas para el sistema de gestión de contactos (prueba_sgc_final.py)
